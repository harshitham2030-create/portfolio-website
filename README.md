# Portfolio Website

This is a single-page portfolio website created as part of an academic project.

## Project Description
The project is a beginner-level portfolio website developed using front-end technologies. It displays basic information about the student and includes simple interactivity using JavaScript.

## Technologies Used
- HTML – for structure
- CSS – for styling
- JavaScript – for basic interactivity

## Backend and Database
This project focuses mainly on front-end development.  
Backend and database concepts are represented conceptually for learning purposes.  
Due to free hosting limitations on GitHub Pages, backend and database integration is not deployed.

## Hosting
The website is hosted using GitHub Pages (free hosting).
